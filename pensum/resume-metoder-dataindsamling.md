# Resume: 2. dag Metoder til dataindsamling - praksis

## Kilde
IBA Erhvervsakademi Kolding, undervisningsmateriale, 45 sider.

## Centrale begreber og modeller

### Undersøgelsesdesign (Møller & Hvid, 2012)
Fire lag:
1. **Verdensbillede** — positivisme / fænomenologi / konstruktivisme
2. **Undersøgelsesstrategier** — kvalitative / kvantitative / kombinerede
3. **Undersøgelsesmetoder** — spørgsmål, dataindsamling, analyse, tolkning, validering
4. **Undersøgelsesdesign** — den samlede ramme

### Tre dataindsamlingsmetoder

**Observation**
- Deltagende / ikke-deltagende
- Mikro (interaktioner) / Makro (arbejdsgange, processer)
- Struktureret / ikke-struktureret
- Kilde: Sunesen (2020)

**Interview (Kvale & Brinkmann)**
- Kvales 7-trins model: Tematisering → Design → Interview → Transkribering → Analyse → Verificering → Rapportering
- Struktureret / semistruktureret / ustruktureret
- Interviewguide som styringsredskab

**Spørgeskema**
- Lukket / halvåbent / åbent
- Likert-skala
- Fejlkilder: Andersen (2019)

### Databehandling
- Transkription (ordret)
- Referatmetoden
- Meningskondensering
- Meningskategorisering / kodning (træstruktur)

### Vurderingskriterier

| Kvantitativ (positivisme) | Kvalitativ (konstruktivisme) |
|---|---|
| Intern validitet | Credibilitet |
| Ekstern validitet | Transferabilitet |
| Reliabilitet | Dependabilitet |
| Objektivitet | Confirmabilitet |
Kilde: Heldberg (1997)
